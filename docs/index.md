# SuperBroke Guide

Welcome to the SuperBroke documentation.

## Quick Start

1. Open the petition form in the main app.
2. Complete each required section in the filing checklist.
3. Verify progress reaches **Ready to File!**
4. Prepare your filing packet.

## Build These Docs

Install MkDocs and run the local docs server:

```bash
pip install mkdocs
mkdocs serve
```

By default, the guide will be available at `http://127.0.0.1:8000/`.
